# touchsprite-extension

这是一个给触动精灵(TouchSprite)开发者开发提供帮助的扩展。

## 使用说明

使用扩展前请先输入开发者AccessKey。

快捷键 **Ctrl+Shift+P** 调出输入框，以下输入命令均在此处输入。


### 激活扩展

> 打开工程文件夹的lua将自动激活扩展  
> Ts: StartServer

### 连接设备

> Ts: Connect

### 获取设备状态

> Ts: GetStatus

### 屏幕截图

> Ts: GetPicture

### 运行项目

> Ts: RunProject  
快捷键 Ctrl+Shift+F9

### 停止运行

> Ts: StopProject  
快捷键 Ctrl+Shift+F10
